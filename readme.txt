# Htoi San Aung - Professional Portfolio Website

## Overview
This is a professional portfolio website for Htoi San Aung, a Psychologist and Clinical Hypnotherapist with over 6 years of experience in mental health and psychosocial support (MHPSS). The website showcases professional experience, skills, services, and contact information.

## Website Structure

### Main Sections:
1. **Header/Navigation** - Smooth scrolling navigation menu
2. **Intro Section** - Personal introduction with profile image and social links
3. **About Section** - Professional background, experience, and skills
4. **Works Section** - Portfolio of services and specializations
5. **Numbers Section** - Key statistics and achievements
6. **Footer/Contact** - Contact information and social media links

## Features

### Interactive Elements:
- Smooth scrolling navigation
- Mobile-responsive design
- Preloader animation
- Image gallery with lightbox functionality
- CV download functionality
- Social media integration

### Key Functionalities:
- **CV Download**: Users can download professional CV
- **Contact Forms**: Direct messaging capability
- **Social Media Links**: Facebook, Instagram, LinkedIn, YouTube
- **Portfolio Gallery**: Showcases different professional services
- **Statistics Counter**: Displays professional achievements
